# /setup

First-time initialization. Run this once, right after scaffolding the vault.

What it does:

1. Registers the qmd collections and contexts (`wiki`, `raw-sources`, `human`, `daily-notes`).
2. Generates local vector embeddings for every markdown file.
3. Writes the index to the path you chose during scaffolding (default `~/.cache/qmd/index.sqlite`).

The first run downloads ~2 GB of GGUF embedding models — this happens once, then is cached across vaults.

Scope: vault-local, installed at `.claude/skills/setup/SKILL.md`. Source: [template/.claude/skills/setup/SKILL.md](https://github.com/jessepinkman9900/claude-second-brain/blob/main/template/.claude/skills/setup/SKILL.md).
