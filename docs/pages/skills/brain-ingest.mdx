# /brain-ingest

Ingest a new source into the wiki.

Drop a file into `raw-sources/articles/`, `raw-sources/pdfs/`, or `raw-sources/personal/`, then run `/brain-ingest`. Claude will:

1. **Read** the source in full.
2. **Summarize** it in 3–5 bullets and ask you what matters most.
3. **Create** a `wiki/sources/[slug].md` page with abstract, key claims, notable quotes, and your synthesis note.
4. **Update** or create topic/entity pages that the source touches, integrating it as prose (not copy-paste) with source attribution.
5. **Flag contradictions** with `[!WARNING]` callouts on any page where the new source disagrees with existing claims.
6. **Log** the ingest in `wiki/log.md`.

Scope: global skill, installed at `~/.claude/skills/brain-ingest/SKILL.md`. Paths resolve at call time via `npx claude-second-brain path` and `npx claude-second-brain qmd` — both read `~/.claude-second-brain/config.toml` to find the default brain.
